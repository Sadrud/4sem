Подробное описание

Основной файл, в котором содержится алгоритм для поиска выпуклой оболочки.

В этом файле также содержатся вспомагательные методы для работы основного алгоритма. Есть метод для печати получаемых данных с сервера в файл.

int  n

Основной метод для поиска выпуклой оболочки множества точек. Функция получает массив точек плоскости, сортирует из по положению на горизонтальной оси. Берется каждая точка из массива и вычисляют ее положение с помощью вмпомагательных функций. Подходящие точки добавляются в многоугольник.

Аргументы
[in]pts[],nУказатель на массив точек, количество точек.
Возвращает
Многоугольник, являющийся выпуклой оболочкой множество точек.

◆ leftToRightCmp()

Point *  b 
)

Компаратор дл точек Сравнение точек происходит по их положению на горизонтальной оси, а затем уже по вертикальной (если по горизонтальной одинаковы)

Аргументы
[in]a,bУказатели на две точки
Возвращает
-1 если первая точка левее второй (если одинаковы по положению на горизонтальной оси, то если первая ниже второй), 1 в противном случае, 0 если это одна и та же точка.
