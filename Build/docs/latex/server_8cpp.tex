\doxysubsection{Подробное описание}
Основной файл сервера для вычисления выпуклой оболочки множества точек. 

Этот файл содержит реализацию HTTP-\/сервера, который принимает POST-\/запросы с набором точек и возвращает их выпуклую оболочку в формате JSON.
